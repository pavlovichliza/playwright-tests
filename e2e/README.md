# Playwright Test Automation

Automated UI tests for [Saucedemo](https://www.saucedemo.com) built with Playwright and TypeScript.

## Test Coverage

- Login: positive and negative scenarios
- Product page: add to cart, button state verification
- Cart: product validation
- Sorting: price low to high verification
- Checkout: full e2e flow

## How to Run

Install dependencies:
\```
npm install
\```
Run all tests:
\```
npx playwright test
\```
Run specific test:
\```
npx playwright test checkout.spec.ts
\```
